# Change-Console

改变 console 行为

| 文件名        | 文件描述                                                     |
| ------------- | ------------------------------------------------------------ |
| index.js      | 非babel插件模式，使用传统parse、traverse、generate的方式改变代码 |
| plugin.js     | babel插件模式                                                |
| mainPlugin.js | babel插件模式，使用@babel/core提供的api进行转换              |
